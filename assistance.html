Questions fréquentes

Je ne vois pas de données dans l'app ou les widgets

Vérifiez que vous avez accordé les autorisations HealthKit. Allez dans Réglages > Santé > Accès aux données > Pulsos+ et activez les catégories souhaitées.

Les widgets ne se mettent pas à jour

Les widgets iOS se mettent à jour selon la fréquence déterminée par le système. Ouvrez l'application pour forcer une mise à jour. Si le problème persiste, supprimez le widget et rajoutez-le.

Le rapport PDF ne montre pas certaines données

Le rapport n'inclut que les données disponibles dans HealthKit pour la période sélectionnée.

Comment restaurer mon achat

Ouvrez Pulsos+ et cherchez l'option « Restaurer les achats ». La restauration est automatique avec le même Apple ID.

Comment demander un remboursement

Les remboursements sont gérés par Apple : reportaproblem.apple.com

Appareils compatibles

Pulsos+ nécessite un iPhone avec iOS 17+. Pour la fréquence cardiaque, l'oxygène ou le sommeil, une Apple Watch est nécessaire. Compatible avec les balances connectées et tensiomètres sync avec Apple Santé.
