Perguntas frequentes

Não vejo dados no aplicativo ou nos widgets

Verifique se concedeu permissões de leitura ao HealthKit. Vá a Ajustes > Saúde > Acesso a dados > Pulsos+ e ative as categorias desejadas.

Os widgets não atualizam

Os widgets do iOS são atualizados conforme a frequência do sistema. Abra o aplicativo para forçar uma atualização. Se o problema persistir, remova o widget e adicione-o novamente.

O relatório PDF não mostra alguns dados

O relatório inclui apenas dados disponíveis no HealthKit para o período selecionado.

Como restaurar minha compra

Abra o Pulsos+ e procure a opção “Restaurar compras”. A restauração é automática com o mesmo Apple ID.

Como solicitar reembolso

Os reembolsos são geridos pela Apple: reportaproblem.apple.com

Dispositivos compatíveis

Pulsos+ requer um iPhone com iOS 17+. Para frequência cardíaca, oxigênio ou sono, é necessário um Apple Watch. Compatível com balanças inteligentes e medidores de pressão sincronizados com Apple Saúde.

Não encontrou o que procura?

Escreva para david.margalef@me.com. Responderemos em até 48 horas.
